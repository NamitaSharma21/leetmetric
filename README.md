# 🚀 LeetMetric

A simple and clean web app that displays LeetCode user statistics in a visually appealing way.

## 🔥 Features

- 📊 Fetches user data using API
- 🟢 Shows Easy, Medium, Hard problem progress
- 🎯 Circular progress indicators
- 💡 Clean and modern UI
- ⚡ Fast and responsive

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- REST API

## 🚀 How to Use

1. Enter your LeetCode username
2. Click on "Search"
3. View your stats instantly

## ⚠️ Note

- This project uses a public API
- Some fields like acceptance rate may not always be available
- Demo data may be used for UI showcase
